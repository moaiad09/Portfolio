# Portfolio React (Vite)

This is a lightweight React portfolio scaffold using Vite.

## Quick start
1. Install dependencies:
   ``npm install``
2. Run dev server:
   ``npm run dev``
3. Open displayed URL (http://localhost:5173)

Features:
- Arabic + English translations (switch in header)
- RTL support for Arabic
- Light/Dark theme toggle (stored in localStorage)
- Canvas particle background
- Sections: Home, About, Projects, Experience, Contact

Customize `src/locales/*.json` and `src/App.jsx` to add your content.
